# きもの屋五一〇 公式サイト

## 概要
進級制作として、岐阜県・愛知県に店舗を構える呉服店「きもの屋五一〇」の公式Webサイトを制作しました。
長年使用されていた既存サイトをリニューアルし、今後も継続して活用できるよう、新たな機能を取り入れたサイトを目指しました。

## 使用技術
- HTML
- CSS
- JavaScript
- WordPress

## 工夫した点
- 呉服店らしい和の雰囲気を意識したデザイン・配色
- スマートフォン・タブレットにも対応したレスポンシブデザイン
- クライアントの要望を反映しつつ、ユーザーの利便性を意識した設計
- PageSpeed Insightsを意識し、画像サイズや形式、コードの最適化を実施
- WordPressを導入し、お知らせを管理画面から簡単に更新できるようにした
- BASEを利用し、オンラインストアを開設

## 苦労した点
- デザイン性と表示速度の両立に苦労しましたが、画像の圧縮や読み込み順を意識したコーディングを行うことで改善しました。
- クライアントの要望が自分の想定していたデザインと異なる場合があり、その都度ヒアリングを行いながらデザインを練り直すことに苦労しました。

## 今後の改善点
- アクセシビリティへの配慮（alt属性の設定やコントラストの改善）
- AIに頼った部分もあったため、今後はより自力で実装できるよう理解を深めたい
