# きもの屋五一〇 公式サイト

岐阜県・愛知県に店舗を展開する老舗呉服店「きもの屋五一〇」の公式サイトをリニューアルしました。
これまでのサイトで感じていた使いづらさを改善し、伝統の雰囲気を大切にしながら、今の時代でも使いやすいサイトを目指しています。

## 概要
長年使われてきた既存サイトを一新し、
**情報を更新しやすくすること** と **ECサイトへの導線を分かりやすくすること** を重視して制作しました。
これから先も長く使い続けられるサイトになるよう設計しています。

- **URL:** [(https://sn-50.boo.jp/kimonoyagotou510/)]
- **制作期間:** 2025年9月 〜 2026年1月（約5ヶ月）
- **制作形態:** 進級制作（個人制作）

## 工夫した点

### 1. 和の雰囲気を活かしたデザイン
呉服店らしい上品な印象を大切にし、ロゴの紺色を基調とした落ち着いた色合いと余白を使ったレイアウトを取り入れました。

### 2. 更新しやすいWordPress設計
お知らせや商品情報を、管理画面から簡単に更新できるようにしました。
HTMLの知識がなくても操作できるよう、入力項目を分かりやすく整理しています。

### 3. 表示速度と見やすさへの配慮
- **表示速度:** 画像をWebP形式にし、JavaScriptの読み込み方法を工夫することで、高速表示を実現しました。
- **レスポンシブ対応:** PC・タブレット・スマートフォンのどの画面でも、快適に見られるよう調整しています。

---

## 苦労した点と解決策
- **デザインと表示速度の両立**
  着物の写真は高画質な分、重くなりやすいため、
  画像の圧縮や遅延読み込みを取り入れて、見た目と速さのバランスを取りました。

- **クライアントとのやりとりを通した制作**
  実際にクライアントの方と打ち合わせを行い、要望を聞きながら制作を進めました。
  いただいた要望をそのまま反映するだけでなく、使いやすさや更新のしやすさも考え、
  クライアントの意図を大切にしつつ、Webサイトとしての利便性を保つことを意識しました。
